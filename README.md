# Digital Gov Cypress Automation

##  Description
End-to-end automation project for testing the gov.il website using Cypress.

##  Test Coverage
- Search functionality tests (header search)
- API validation (status code, response)
- Intercept tests (mocking and modifying responses)

##  Technologies
- Cypress
- JavaScript
- Page Object Model (POM)

##  Project Structure

  e2e/
  pages/
  fixtures/
  support/
